# 社区共享工具服务台

基于 Svelte + Vite + TypeScript + Tailwind CSS 构建的社区共享工具借用管理服务台。

## 项目简介

为社区居民提供工具的在线查询、预约和借用服务，同时为服务站工作人员和维修志愿者提供管理入口。围绕电钻、梯子、推车、绿化剪、清洗机等工具的规格、可借日期、借用人、押金、归还照片、损坏部位和维修进度展开管理。

## 原始需求

> 做一个社区共享工具借用页面，Svelte + Vite 适合把工具库存、预约日历、押金状态和维修队列做成轻量服务台。页面围绕电钻、梯子、推车、绿化剪、清洗机等工具的规格、可借日期、借用人、押金、归还照片、损坏部位和维修进度展开。居民要能看到工具是否可借、何时归还、是否需要培训；服务站登记借出归还和押金退还；维修志愿者处理损坏工具并更新可用状态。页面要让可借、已借、待检、维修中和报废状态一眼分清。

## 技术栈

- **前端框架**: Svelte 4 + TypeScript
- **构建工具**: Vite 5
- **样式方案**: Tailwind CSS 3
- **状态管理**: Svelte 内置 store
- **图标库**: lucide-svelte

## 核心功能

### 用户角色

| 角色 | 功能权限 |
|------|----------|
| 社区居民 | 浏览工具库存、查看可借日期、查看借用状态、了解培训要求 |
| 服务站管理员 | 登记借出/归还、管理押金退还、更新工具状态 |
| 维修志愿者 | 处理损坏工具、更新维修进度、标记工具可用/报废 |

### 功能模块

1. **工具库存看板**: 工具卡片列表、状态筛选、规格详情展示
2. **预约日历视图**: 工具借用时间表、可借日期标注、归还日期显示
3. **押金管理面板**: 押金状态追踪、退还记录、借用人信息
4. **维修队列管理**: 损坏工具列表、维修进度更新、报废标记
5. **全局状态指示器**: 可借/已借/待检/维修中/报废 五态一眼分清

### 状态色彩体系

| 状态 | 颜色 | 色值 |
|-----|------|------|
| 可借 | 翠绿 | #22C55E |
| 已借 | 深蓝 | #3B82F6 |
| 待检 | 琥珀 | #F59E0B |
| 维修中 | 赤红 | #EF4444 |
| 报废 | 深灰 | #6B7280 |

## 启动方式

### 前置要求

- Node.js >= 18
- npm 或 pnpm 包管理器

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务

```bash
npm run dev
```

访问地址：http://localhost:5175

#### 3. 类型检查

```bash
npm run check
```

#### 4. 生产构建

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

#### 5. 预览生产构建

```bash
npm run preview
```

## Docker 一键启动

### 前置要求

- Docker >= 20.10
- Docker Compose >= 2.0

### 启动步骤

#### 1. 构建并启动服务

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:5175

#### 2. 停止和清理服务

```bash
docker compose down
```

## 注意事项

- 项目使用 Mock 数据进行演示，无真实后端服务
- 数据状态仅保存在浏览器内存中，刷新页面后恢复初始状态
- 所有操作按钮均为演示用，可根据实际需求对接后端 API
